# 网页小游戏

一个使用 HTML、CSS 和 JavaScript 编写的网页小游戏集合。

## 游戏列表

### 1. 贪吃蛇 🐍

经典的小游戏，控制蛇吃掉食物来获得分数。

**文件：** `snake_game.html`

**功能特点：**
- 深色科技风界面，荧光绿配色
- 渐变色蛇身，圆形食物设计
- 实时显示得分和最高分（本地存储）
- 难度递增（每50分加速）
- 游戏结束后显示最终得分

**操作方式：**
| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制蛇的移动方向 |
| W A S D | 键盘方向控制 |
| 空格键 | 暂停/继续游戏 |
| R | 重新开始 |

---

### 2. 五子棋 🎲

经典的双人对战棋类游戏。

**文件：** `gomoku.html`

**功能特点：**
- 木质棋盘，仿真实物棋子
- 黑白双人对战
- 显示最后落子位置
- 支持悔棋功能
- 获胜动画提示

**操作方式：**
- 点击棋盘交叉点放置棋子
- 黑白轮流下棋
- 率先连成五子获胜

---

## 如何运行

### 方法一：直接打开
在文件资源管理器中找到 HTML 文件，双击即可在浏览器中打开。

### 方法二：本地服务器
```bash
python -m http.server 8080
```
然后在浏览器中访问：
- 贪吃蛇：http://localhost:8080/snake_game.html
- 五子棋：http://localhost:8080/gomoku.html
