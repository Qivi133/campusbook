## 4. 功能需求详情

### 4.1 用户系统

#### 4.1.1 用户注册

**功能描述**：新用户创建账户

**必填字段**：
- 用户名（2-50字符，唯一）
- 邮箱（有效格式，唯一）
- 密码（至少6位）

**可选字段**：
- 手机号

**业务流程**：
```
用户填写信息 → 验证格式 → 检查唯一性 → 创建账户 → 自动登录 → 跳转首页
```

#### 4.1.2 用户登录

**必填字段**：
- 用户名
- 密码

**业务流程**：
```
用户输入凭证 → 验证账号密码 → 创建会话 → 跳转首页
```

#### 4.1.3 用户登出

**业务流程**：
```
点击登出 → 清除会话 → 跳转首页
```

---

### 4.2 书籍系统

#### 4.2.1 发布书籍

**必填字段**：
- 书名（最多200字符）
- 分类（预设分类）
- 价格（数字，单位：元）

**可选字段**：
- 作者
- ISBN
- 新旧程度（下拉选择）
- 描述（文本）
- 图片（支持jpg/png/gif）

**预设分类**：
- 教材
- 文学
- 专业课
- 考研
- 公务员
- 其他

**业务流程**：
```
登录用户 → 填写书籍信息 → 上传图片（可选） → 提交 → 创建记录 → 显示详情页
```

#### 4.2.2 浏览书籍

**展示内容**：
- 书籍图片
- 书名
- 作者
- 价格
- 分类标签
- 状态标签

**分页**：每页12本

**排序**：最新发布（默认）

#### 4.2.3 书籍详情

**展示内容**：
- 书籍大图
- 书名、作者、价格
- 分类、ISBN、新旧程度
- 卖家用户名
- 发布时间
- 详细描述

**根据用户角色显示操作**：
- 游客：仅查看
- 卖家：编辑、删除
- 买家：加入购物车

#### 4.2.4 编辑书籍

**权限**：仅书籍所有者

**业务流程**：
```
访问编辑页 → 修改信息 → 保存 → 更新数据库 → 显示详情页
```

#### 4.2.5 删除书籍

**权限**：仅书籍所有者

**业务流程**：
```
点击删除 → 确认框 → 确认 → 删除记录 → 跳转首页
```

---

### 4.3 搜索系统

#### 4.3.1 关键词搜索

**搜索范围**：书名、作者

**匹配方式**：模糊匹配，不区分大小写

#### 4.3.2 分类筛选

**筛选方式**：选择分类，显示该分类在售书籍

---

### 4.4 交易系统

#### 4.4.1 加入购物车

**业务规则**：
- 仅登录用户
- 不能购买自己的书
- 书籍必须在售
- 不能重复添加

**业务流程**：
```
书籍详情页 → 点击加入购物车 → 验证状态 → 创建记录 → 跳转购物车
```

#### 4.4.2 查看购物车

**展示内容**：
- 书籍图片、书名
- 卖家、价格
- 状态
- 移除按钮

**统计**：总价合计

#### 4.4.3 下单购买

**业务流程**：
```
购物车页 → 点击下单 → 创建订单 → 更新书籍状态 → 清空购物车 → 跳转订单页
```

**业务规则**：
- 购物车不能为空
- 只会为在售书籍创建订单

#### 4.4.4 订单查看

**展示内容**（购买订单）：
- 书籍信息
- 卖家、价格
- 状态
- 时间

**订单状态**：
- pending（待确认）
- completed（已完成）
- cancelled（已取消）

## 6. 数据模型

### 6.1 数据库：SQLite

### 6.2 数据表

#### User（用户）
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK | 主键 |
| username | String(50) | Unique, Not Null | 用户名 |
| email | String(100) | Unique, Not Null | 邮箱 |
| password_hash | String(200) | Not Null | 密码哈希 |
| phone | String(20) | Nullable | 手机号 |
| created_at | DateTime | Default | 创建时间 |

#### Book（书籍）
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK | 主键 |
| title | String(200) | Not Null | 书名 |
| author | String(100) | Nullable | 作者 |
| isbn | String(20) | Nullable | ISBN |
| category | String(50) | Nullable | 分类 |
| price | Float | Not Null | 价格 |
| description | Text | Nullable | 描述 |
| image_url | String(200) | Nullable | 图片URL |
| condition | String(20) | Nullable | 新旧程度 |
| seller_id | Integer | FK → User | 卖家ID |
| status | String(20) | Default: available | 状态 |
| created_at | DateTime | Default | 发布时间 |

#### Order（订单）
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK | 主键 |
| buyer_id | Integer | FK → User | 买家ID |
| book_id | Integer | FK → Book | 书籍ID |
| total_price | Float | Not Null | 总价 |
| status | String(20) | Default: pending | 状态 |
| contact_message | String(500) | Nullable | 留言 |
| created_at | DateTime | Default | 创建时间 |

#### CartItem（购物车）
| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | Integer | PK | 主键 |
| user_id | Integer | FK → User | 用户ID |
| book_id | Integer | FK → Book | 书籍ID |
| created_at | DateTime | Default | 添加时间 |
